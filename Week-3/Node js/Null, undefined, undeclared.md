## null
`null` is an empty or non-existent value.  
`null` must be assigned.
```
let a = null;
console.log(a);
// null
```
## undefined
Undefined most typically means a variable has been declared, but not defined.    
```
let b;
console.log(b);
// undefined
```
## undeclared
It means a variable is not at all declared. 
```
console.log(c);
```
